# WooCommerce User Discounts

**Descuentos de Usuario para WooCommerce** es un plugin que permite asignar descuentos personalizados a usuarios basados en una taxonomía específica. Esta herramienta es ideal para tiendas que desean ofrecer precios especiales a ciertos grupos de usuarios o miembros.

## Características

- Asigna descuentos personalizados a usuarios individuales.
- Utiliza una taxonomía personalizada para gestionar y categorizar los descuentos.
- Aplica descuentos a productos simples y variaciones.
- Controla la visualización de precios con descuento en productos variables.
- Fácil de instalar y configurar.

## Instalación

1. Descarga el archivo ZIP del plugin.
2. Ve a **WordPress Dashboard > Plugins > Añadir nuevo**.
3. Haz clic en **Subir Plugin** y selecciona el archivo ZIP.
4. Activa el plugin.
5. Ve a la sección de taxonomías y comienza a asignar descuentos a los usuarios.

## Uso

1. Una vez instalado, ve a **Usuarios** en tu panel de WordPress.
2. Selecciona un usuario y asigna una taxonomía de descuento.
3. Configura el porcentaje de descuento y otras opciones relacionadas.
4. Los precios se ajustarán automáticamente en la tienda según las configuraciones.
